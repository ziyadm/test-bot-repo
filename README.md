# setup
create a [.env] file one level up from where this [README.md] file is and fill it with the required secrets
```
$ make setup
```


# build
```
$ make build
```


# starting work on a new feature
```
$ make feature
<feature-name-separated-by-dashes>
```


# committing changes
```
$ make push
```
